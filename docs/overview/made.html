Neutronium.SPA.Demo

https://github.com/NeutroniumCore/Neutronium.SPA.Demo

Neutronium.SPA.Demo is a scaffold Vue.js reference project that can be used as a starter or reference when building a large project. It includes IoC (example Ninject but easily changeable), navigation with integration with vue-router, internalization with integration with vue-i18n. Built on top of Vuetify.js.
